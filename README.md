# 🏠 1C Infrastructure
Домашний сервер для 1С-разработки на базе **Geekom A9 Max** (Ryzen AI 9 HX 370, Windows 11 Pro).

> **Цель:** Изолированная среда для 1С + мониторинг + автоматические бэкапы + безопасный доступ + **интеграция с локальным мессенджером для бизнес-уведомлений**.

---

## 🧩 Компоненты

### Основные сервисы
| Сервис | Назначение | Порт |
|--------|------------|------|
| **PostgreSQL** | СУБД для 1С:Предприятие | 5432 |
| **1С:Предприятие** | Сервер 1С (dev / test / prod) | — |
| **Обновлятор 1С** | Автоматические бэкапы баз | — |
| **Tailscale VPN** | Безопасный удалённый доступ | WireGuard |

### Инфраструктура и мониторинг
| Сервис | Назначение | Порт |
|--------|------------|------|
| **Portainer** | Управление Docker-контейнерами | 9000 |
| **pgAdmin 4** | Веб-интерфейс PostgreSQL | 5050 |
| **Prometheus** | Сбор метрик | 9090 |
| **Grafana** | Визуализация и алерты | 3002 |
| **VoceChat** | Корпоративный мессенджер | 3001 |
| **Blackbox Exporter** | HTTP-проверки доступности | 9115 |
| **postgres-exporter** | Метрики PostgreSQL | 9187 |
| **cAdvisor** | Метрики контейнеров (CPU/RAM) | 8080 |

### 🔔 Бизнес-интеграция
| Компонент | Назначение | Контекст |
|-----------|------------|----------|
| **ВЧ_ИнтеграцияСVoceChat** | Автоматические уведомления в корпоративный мессенджер при приёме сотрудников (1С:ЗУП 3.1) | Расширение конфигурации, асинхронная очередь, регламентное задание |

**Доступ:** `localhost` или `100.x.x.x` через Tailscale VPN

## 🚀 Быстрый старт

```powershell
# 1. Клонировать репозиторий
git clone https://github.com/VladimirProgrammist1C/1c-infrastructure.git
cd 1C_Infrastructure

# 2. Настроить пароли (обязательно!)
copy .env.example .env
notepad .env # ← изменить пароли и пути

# 3. Запустить все сервисы
docker-compose up -d

# 4. Проверить статус
docker-compose ps # все должны быть "Up (healthy)"
```
