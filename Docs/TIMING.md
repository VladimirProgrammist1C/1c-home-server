### 🗓️ 30.03.2026 — Подключение 1С к PostgreSQL

| Время | Задача | Результат |
|-------|--------|-----------|
| 0:25 | Проблема с локалью кластера | ❌ Первая попытка создать базу через консоль 1С не удалась |
| 0:35 | Пересборка контейнера с `ru_RU.UTF-8` | ✅ `entrypoint.sh` настроен с русской локалью |
| 0:30 | Создание базы через консоль 1С | ✅ База `DemoHRMCorpDemo_bot` открыта |

**Критично:** Кластер PostgreSQL должен быть инициализирован с русской локалью (`ru_RU.UTF-8`). Без этого 1С не сможет создавать базы ни через консоль администрирования, ни через ярлык, ни через Обновлятор!

**Решение в `entrypoint.sh`:**
```bash
export LANG=ru_RU.UTF-8
export LC_COLLATE=ru_RU.UTF-8
export LC_CTYPE=ru_RU.UTF-8
```

### 🗓️ 03.04.2026 — Мониторинг и алерты

| Время | Задача | Результат |
|-------|--------|-----------|
| 0:30 | Выбор стратегии мониторинга | Prometheus + Grafana + Blackbox |
| 0:45 | Настройка cAdvisor | Метрики CPU/RAM контейнеров |
| 1:00 | Настройка Prometheus | Сбор метрик каждые 15-30 сек |
| 1:30 | postgres-exporter | Метрики PostgreSQL (`pg_up`) |
| 1:00 | Blackbox Exporter | HTTP-проверки доступности |
| 0:45 | Настройка алертов в Grafana | 10 правил (критические + предупреждения) |
| 0:30 | Уведомления VoceChat | Шаблоны с форматированием |

**Проблема 1:** cAdvisor не собирает метрики контейнеров на Docker Desktop + WSL2  
**Решение:** Использовать cAdvisor только для метрик хоста, контейнеры через Blackbox

**Проблема 2:** pgAdmin и Grafana возвращают 401/302 вместо 200  
**Решение:** 
- Grafana: `/api/health` endpoint
- pgAdmin: custom blackbox.yml с кодами 2xx/3xx/401/403

**Проблема 3:** Уведомления приходят в формате JSON (не читаемо)  
**Решение:** Content-Type: `text/plain`, шаблон в Title и Message

**Итог:** ✅ 9 из 10 алертов работают, уведомления читаемые
