## 🎯 Назначение
Расширение предназначено для интеграции **1С:ЗУП 3.1** с локальным мессенджером **VoceChat**. 
При проведении документа «Прием на работу» система автоматически формирует и отправляет задачу для системного администратора или HR-менеджера в указанный канал VoceChat.

**Зачем это нужно:**
* ⚡ **Мгновенные уведомления:** Администратор узнает о новом сотруднике сразу, без ожидания отчетов.
* 🛠 **Автоматизация:** Исключается ручной труд по отправке писем/сообщений в чаты.
* 📊 **Контроль:** Все задачи фиксируются в регистре 1С с историей отправок и статусами.
* 🔄 **Надёжность:** Асинхронная очередь с повторными попытками при ошибках сети.

## 🚀 Быстрая установка
1. **Скопируйте файл расширения:**
   ```
   build/ВЧ_ИнтеграцияСVoceChat_v1.0.cfe
   ```
2. **Установите в 1С:**
   * Откройте Конфигуратор → **Конфигурация → Расширения конфигурации → Добавить**.
   * Выберите файл `.cfe`.
   * Нажмите **F7** для обновления конфигурации базы данных.
3. **Настройте интеграцию:**
   * В 1С откройте общую форму **«Настройки интеграции с VoceChat»**.
   * Включите переключатель **«Использовать интеграцию»**.
   * Укажите: `URL сервера`, `Порт`, `BotAPIKey`, `ID канала`, `Лимит попыток`, `Таймаут HTTP`.
4. **Проверьте:**
   * Проведите новый или непроведенный документ «Прием на работу».
   * Убедитесь, что уведомление появилось в VoceChat.
